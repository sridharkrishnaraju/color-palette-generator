# Color Palette Generator

Pick a base colour and get a five-step tints-and-shades palette with hex codes. A free, zero-dependency Web Component you can drop into any site.

```html
<script src="palette-generator.js"></script>
<palette-generator></palette-generator>
```

## What it does

A colour palette generator turns a single brand colour into a set of related colours — lighter **tints** for backgrounds and hover states, and darker **shades** for text and borders. It works by converting the base colour to HSL and stepping the lightness up and down, which keeps the hue consistent and the result harmonious.

## When you'd use it

Building a UI colour scale from a brand colour, or finding background/border variants that still feel on-brand.

See the [Usage & API guide](usage.md) for framework examples, or the [guide](guide.md) for tips.
